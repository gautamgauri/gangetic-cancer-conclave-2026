Gangetic Cancer Conclave — Chapter II (Static Website Package)

What’s inside
- Standalone HTML pages (index.html, about.html, program.html, faculty.html, abstracts.html, registration.html, venue-travel.html, sponsors.html, committee.html, contact.html, policies.html)
- abstract-ladder-scaffold.html + tariff-table-scaffold.html
- assets/style.css + assets/site.js
- assets/AGENDA-2.pdf (brochure copy from upload)
- assets/img/* placeholder headshots

How to preview locally
- cd into extracted folder
- python3 -m http.server 8080
- open http://localhost:8080

Replace photos
- Replace assets/img/*.jpg/png with approved headshots (keep file names the same)
- Recommended: square crop (1:1), 800x800 or 1000x1000, ~150–300KB
